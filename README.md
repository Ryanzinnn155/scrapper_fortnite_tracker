<h1>WebScrapping Fortnite Tracker</h1>

<h2>How Scrapper works</h2>
<p>
Scrapper starts by starting a selenium session by opening a tab in "Google Chrome" and then entering the page that was 
provided the URL. Then he starts collecting information from the score table to the parameter he was told to go to!
</p>

<h3>How to use it</h3>
<p>
To use this Scrapper it is very simple, just download the document with the code or simply copy it, then you will go to the "Fortnite Tracker Events" website and you will only need to get its URL and specify the parameter of which page it will Scrap to. .
  With everything it will return the following information, "Rank", "Name", "Points", "Matches", "Wins", "Avg Elims" and "Avg Places".
</p>
